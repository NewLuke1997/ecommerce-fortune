# 🏮 跨境电商运势签

每日跨境电商运势指南 — 选择平台、输入信息，获取今日专属电商运营建议！

## ✨ 功能

- 🛒 **八大平台支持** — Amazon / Shopee / Lazada / TikTok Shop / Temu / SHEIN / eBay / AliExpress
- 🎯 **六大运营维度** — 广告投放 / 库存管理 / 定价策略 / 选品建议 / 物流运势 / 客服沟通
- 🍀 **每日开运指南** — 幸运颜色、数字、吉时、宜忌
- 📱 **移动端适配** — 完美支持手机浏览
- 📤 **一键分享** — 复制运势分享给朋友

## 🚀 使用

直接打开 `index.html` 即可使用，纯前端项目，无需后端。

## 📂 项目结构

```
├── index.html    # 主页面 - 抽签运势
├── about.html    # 关于我们 - 微信二维码
├── style.css     # 样式
└── script.js     # 运势逻辑
```

## 🎨 设计风格

中国风 + 暗色主题，配合浮动粒子动效，营造神秘的运势氛围。
